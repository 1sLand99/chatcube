# Changelog

## v1.0.4 (2026-03-04)

这一版主打「更顺手 + 更稳定」：模型学会分组列队、输出不抢你滚动、设置弹窗更可靠。

### ✨ 新增 / 改进

- 获取模型支持分组显示（找模型更快）
- 支持聊天字体大小调节（看得更舒服）
- 对话列表更紧凑：左右留白优化、左滑操作区更克制，FAB 更好点

### 🧠 体验优化

- 模型输出内容时不再影响用户上下滑动查看内容
- 服务商界面键盘避让，输入不再被挡住

### 🛠 稳定性与修复

- 设置页弹窗逻辑优化，更加稳定可靠
- 优化异常情况报错提示，更清晰可读
- 优化聊天界面大量消息时 jump 到底的逻辑，不再闪退
